Weight Lifting Journal

    Coding Languages used

  • HTML
  • CSS
  • LESS

Our job for this project was to create a weight lifting journal for anyone who wants to keep track with their training. My responsibilities were to create the landing page using mainly HTML & CSS. The website also required strong responsiveness for phone viewers (500px), tablet viewers (800px), and desktop viewers(1000px).

The most enjoyble exprerience on this project was working with flexbox. Flexbox made creating thee landing page much easier, from structuring my navigational bar to organizing my divs. Flexbox is a life saver. Along with flexbox I used mixins and nesting to help me organize my styling with CSS.

Chef Portifilo

    Coding Languages used

  • React
  • CSS
  • HTML

Chef Portifilo is a website my team created to offer a list of a recipes for general users and professional chefs. The website allows general users to view a list of recipes. For chefs they to can view a list of recipes while also having the ability to create an account where they can create, edit, and delete their own recipes. My job for this website was to build a state management system to hold data and transport that data to the backend database. I created a global state, a component tree state, and and an individual component state. I also had to make HTTP request (received endpoints from the backend guys) using the CRUD Methods and create a token authentication for the login and sign in form I created.
